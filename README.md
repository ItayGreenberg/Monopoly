\# Monopoly



A C# Monopoly game with separate client and server.



\## Project structure



\- `MonopolyClient` - client application

\- `MonopolyServer` - server application

\- `Executables` - compiled binaries (not needed to build from source)

\- `Pictures` - images and assets

\- `diagram` - design documentation and diagrams
